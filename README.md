# ♟️ Chess Coach AI

A Gen AI powered chess coach built with Google Gemini API and Streamlit.

## Features
- Ask any chess question and get expert answers
- Built with Google Gemini 2.5 Flash
- Simple and clean chat interface

## Tech Stack
- Python
- Google Gemini API
- Streamlit

## How to Run
pip install -r requirements.txt
streamlit run chess_app.py
